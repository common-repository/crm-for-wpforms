CRM for WPForms Contact Form

== Description ==

CRM for WPForms Contact Form use Cloodo API. Plugin will register an account automatic after you see first screen on erp.cloodo.com. All your data will be saved here. Your view will show on worksuite.cloodo.com and connect.cloodo.com.

Cloodo Privacy Policy: https://cloodo.com/page/privacy-policy
